# meinprojekt
#katzengewichtstracker

Projekt für Web Architecture. Zeigt den Gewichtsverlauf meiner Katze als Diagramm.

Ich habe mich bewusst für Vite entschieden, da die Website interaktiv ist und einen eher app-artigen Ablauf hat. Eine schnelle Client-Interaktion liegt im Fokus. Next.js würde nur dann Vorteile bringen, wenn ich öffentlich auffindbare Inhalte hätte, die von Suchmaschinen sauber idexiert werden sollten. Trotz meines Community-Forums entscheide ich mich für Vite, weil das Forum hauptsächlich innerhalb der App genutzt wird und Interaktivität wichtiger ist als die Suchmaschinenoptimierung.
